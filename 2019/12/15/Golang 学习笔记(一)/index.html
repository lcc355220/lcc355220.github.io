init 函数

  • 在 main 被执行前，所有依赖的 package 的 init 方法都会被执行
  • 不同包的 init 函数按照包导入的依赖关系决定执行顺序
  • 每个包可以有多个 init 函数
  • 包的每个源文件也可以有多个 init 函数，这点比较特殊

Java 线程和 Go 协程的区别

1.创建时默认的 stack（栈）的大小

  • JDK5以后 Java Thread stack 默认为1M
  • Groutine 的 Stack 初始化大小为2K

2.和 KSE (Kernel Space Entity) 的对应关系

  • Java Thread 是 1：1
  • Groutine 是 M:N

Context

  3. 当前 Context 被取消时，基于他的子 context 都会被取消
